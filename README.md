# 🤖 Agentic PR Reviewer (v2.1)

An autonomous AI-powered code review system that bridges the gap between static code analysis and dynamic UI testing. This repository demonstrates a "single-brain" agentic architecture using **Gemini 2.5 Flash** and **Playwright**.

## 🚀 How It Works

1.  **Trigger**: Triggers on PR updates or a manual `@qwen-code review` comment.
2.  **Analysis**: The agent fetches the PR diff and uses Gemini to reason about the changes.
3.  **Planning**: Gemini generates a custom JSON test plan with interactive steps (click, type, wait).
4.  **Execution**: Playwright executes the plan in a headless browser, capturing a **visual trace** (frame-by-frame screenshots).
5.  **Review**: Gemini synthesizes the test results, console logs, and visual evidence into a high-impact PR review.

## 🛠️ Project Structure

- **`.github/workflows/agentic-pr-review.yml`**: The GitHub Actions orchestrator.
- **`scripts/agentic_reviewer.js`**: The consolidated "Single-Brain" agent script.
- **`index.html`**: The Marketplace web application (the test target).
- **`package.json`**: Dependency management for deterministic builds.

## 🧠 Key Design Strengths

✅ **Full Automation**: No human QA needed for basic regression flows.  
✅ **Behavior-Based Testing**: Tests real UI interactions and states.  
✅ **AI Reasoning**: Dynamically adapts test scenarios based on specific code changes.  
✅ **Evidence-Driven**: Reviews are grounded in logs, errors, and visual screenshots.  
✅ **GitHub Native**: Seamless integration with the PR feedback loop.  

## ⚠️ Current Limitations ("Lacs")

While powerful, the current version has several areas for growth:

❌ **No Visual Understanding**: Currently, the LLM analyzes screenshot *metadata* and logs, but does not yet "see" the raw pixels of the PNGs (Vision Upgrade Pending).  
❌ **Limited Assertions**: Relies on navigation success and console logs rather than explicit `expect()` style UI assertions.  
❌ **Diff Truncation**: Extremely large PRs may have their diffs truncated to fit LLM context windows (max 10k chars).  
❌ **Static Selectors**: While it uses robust IDs, it still relies on a predefined list of "anchor" selectors provided in the prompt.  

## 🔮 Future Roadmap

- [ ] **Multimodal Vision**: Send base64 image data to Gemini for true visual bug detection.
